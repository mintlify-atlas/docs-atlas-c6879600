# Developer Installation

This guide walks you through setting up eStory locally for development, including environment configuration, database setup, and smart contract deployment to Base Sepolia.

<Note>
  **Prerequisites**: Node.js 18+, npm/yarn/pnpm, Supabase account, MetaMask wallet with Base Sepolia testnet ETH, and API keys for AI services.
</Note>

## Prerequisites

### Required Software

- **Node.js 18+** (eStory uses Next.js 15.5.9 with React 19)
- **npm, yarn, or pnpm** (package manager)
- **Git** for cloning the repository

### Required Accounts & API Keys

| Service | Purpose | Get API Key |
|---------|---------|-------------|
| **Supabase** | PostgreSQL database + auth + storage | [supabase.com/dashboard](https://supabase.com/dashboard/project/[your-project]/settings/api) |
| **WalletConnect** | RainbowKit wallet connection | [cloud.walletconnect.com](https://cloud.walletconnect.com/) |
| **Google Gemini** | AI text enhancement + story analysis | [makersuite.google.com/app/apikey](https://makersuite.google.com/app/apikey) |
| **ElevenLabs** | Speech-to-text transcription | [elevenlabs.io/app/settings/api-keys](https://elevenlabs.io/app/settings/api-keys) |
| **Pinata** | IPFS storage for NFT metadata | [app.pinata.cloud/developers/api-keys](https://app.pinata.cloud/developers/api-keys) |
| **Resend** | Transactional emails | [resend.com/api-keys](https://resend.com/api-keys) |
| **Basescan** | Contract verification (optional) | [basescan.org/myapikey](https://basescan.org/myapikey) |

### Base Sepolia Testnet Setup

1. Add Base Sepolia to MetaMask:
   - **Network Name**: Base Sepolia
   - **RPC URL**: `https://sepolia.base.org`
   - **Chain ID**: `84532`
   - **Currency Symbol**: `ETH`
   - **Block Explorer**: `https://sepolia.basescan.org`

2. Get testnet ETH from [Base Sepolia Faucet](https://www.coinbase.com/faucets/base-ethereum-goerli-faucet)

### 4. Set Up Supabase Database

#### Create Supabase Project

1. Go to [supabase.com/dashboard](https://supabase.com/dashboard)
2. Click **"New Project"**
3. Choose organization, name, database password, and region
4. Wait for project provisioning (~2 minutes)

#### Run Database Migrations

Run the migrations in `supabase/migrations/` to create tables:

```bash
cd supabase/migrations
```

Key migrations to apply (in order):

1. **`001_create_weekly_reflections.sql`** - Weekly AI-generated summaries
2. **`002_enable_rls_policies.sql`** - Row Level Security for data privacy
3. **`003_add_oauth_fields.sql`** - OAuth account linking fields
4. **`004_create_habits_tables.sql`** - Habit tracking
5. **`005_create_waitlist_table.sql`** - Waitlist management
6. **`006_create_verified_metrics_tables.sql`** - CRE verification tables

Apply via Supabase SQL Editor or Supabase CLI:

<CodeGroup>

```bash Supabase CLI
# Install Supabase CLI
npm install -g supabase

# Link to your project
supabase link --project-ref YOUR_PROJECT_REF

# Push migrations
supabase db push
```

```bash Manual (SQL Editor)
# Copy-paste each migration file into Supabase SQL Editor
# Run them in order (001 → 002 → 003 → ...)
```

</CodeGroup>

#### Create Storage Bucket

1. Go to **Storage** in Supabase dashboard
2. Click **"New bucket"**
3. Name: `story-audio`
4. Set **Public bucket** to `true` (audio files need public URLs)
5. Click **"Create bucket"**

#### Enable Authentication Providers

1. Go to **Authentication > Providers**
2. Enable **Email** (for wallet auth)
3. Enable **Google OAuth**:
   - Add Google Client ID and Secret from [Google Cloud Console](https://console.cloud.google.com/apis/credentials)
   - Set redirect URL to `https://YOUR_PROJECT.supabase.co/auth/v1/callback`

### 5. Deploy Smart Contracts

eStory uses three Solidity contracts deployed on Base Sepolia:

1. **eStoryToken** (ERC20) - $STORY token with 100M max supply
2. **StoryProtocol** - Tips and paywall payments
3. **StoryNFT** (ERC721) - Story book NFTs with 0.001 ETH mint fee

#### Compile Contracts

```bash
npx hardhat compile
```

This generates ABIs in `artifacts/contracts/`.

#### Deploy to Base Sepolia

<CodeGroup>

```bash Deploy Script
# Deploy all contracts (from scripts/deploy.ts)
npx hardhat run scripts/deploy.ts --network baseSepolia
```

```typescript Deploy Script Code
// scripts/deploy.ts (from source)
import { ethers } from "hardhat";

async function main() {
  const [deployer] = await ethers.getSigners();
  console.log("Deploying with:", deployer.address);

  // 1. Deploy EStoryToken
  const EStoryToken = await ethers.getContractFactory("EStoryToken");
  const token = await EStoryToken.deploy(deployer.address);
  await token.waitForDeployment();
  const tokenAddress = await token.getAddress();
  console.log(`EStoryToken deployed: ${tokenAddress}`);

  // 2. Deploy StoryProtocol
  const StoryProtocol = await ethers.getContractFactory("StoryProtocol");
  const protocol = await StoryProtocol.deploy(tokenAddress, deployer.address);
  await protocol.waitForDeployment();
  const protocolAddress = await protocol.getAddress();
  console.log(`StoryProtocol deployed: ${protocolAddress}`);

  // 3. Deploy StoryNFT
  const StoryNFT = await ethers.getContractFactory("StoryNFT");
  const nft = await StoryNFT.deploy(deployer.address);
  await nft.waitForDeployment();
  const nftAddress = await nft.getAddress();
  console.log(`StoryNFT deployed: ${nftAddress}`);
}

main().catch((error) => {
  console.error(error);
  process.exitCode = 1;
});
```

</CodeGroup>

**Expected Output**:

```
Deploying contracts with the account: 0xYourAddress
EStoryToken deployed to: 0xf9eDD76B55F58Bf4E8Ae2A90a1D6d8d44dfA74BC
StoryProtocol deployed to: 0xA51a4cA00cC4C81A5F7cB916D0BFa1a4aD6f4a71
StoryNFT deployed to: 0x6D37ebc5eAEF37ecC888689f295D114187933342

--- SAVE THESE ADDRESSES ---
Go to 'lib/contracts.ts' and update the addresses now.
```

## Troubleshooting

### Build Errors

**Issue**: `Cannot find module '@/lib/contracts'`

**Solution**: Ensure `tsconfig.json` has path alias:

```json
{
  "compilerOptions": {
    "paths": {
      "@/*": ["./*"]
    }
  }
}
```

### Supabase Connection Errors

**Issue**: `Invalid API key` or `Project not found`

**Solution**: 
1. Verify `NEXT_PUBLIC_SUPABASE_URL` and `NEXT_PUBLIC_SUPABASE_ANON_KEY` in `.env.local`
2. Check Supabase dashboard → Settings → API
3. Restart dev server after changing env vars

### Web3 Connection Issues

**Issue**: `Unsupported chain` or `Chain mismatch`

**Solution**:
1. Check `lib/wagmi.config.ts` includes Base Sepolia:
   ```typescript
   import { baseSepolia } from 'wagmi/chains';
   export const chains = [baseSepolia];
   ```
2. Switch MetaMask to Base Sepolia (Chain ID: 84532)

### Contract Deployment Fails

**Issue**: `Insufficient funds` or `Nonce too high`

**Solution**:
1. Ensure wallet has Base Sepolia testnet ETH (get from faucet)
2. Check `hardhat.config.ts` has correct RPC:
   ```typescript
   networks: {
     baseSepolia: {
       url: "https://sepolia.base.org",
       accounts: [process.env.ADMIN_WALLET_PRIVATE_KEY!]
     }
   }
   ```
3. If nonce error, reset account in MetaMask
