## Prerequisites

Before deploying contracts, ensure you have:

<Steps>
  <Step title="Install Dependencies">
    Install Hardhat and required packages:
    
    ```bash
    cd ~/workspace/source
    npm install --save-dev hardhat @nomicfoundation/hardhat-toolbox
    ```
  </Step>
  
  <Step title="Get Testnet ETH">
    Obtain Base Sepolia ETH from the Coinbase faucet:
    
    [Base Sepolia Faucet](https://www.coinbase.com/faucets/base-ethereum-goerli-faucet)
    
    <Note>
      You'll need ~0.1 ETH for deployment gas costs and contract interactions.
    </Note>
  </Step>
  
  <Step title="Configure Environment">
    Create `.env.local` with deployment keys:
    
    ```bash
    # Admin wallet private key (DO NOT COMMIT)
    ADMIN_WALLET_PRIVATE_KEY=0x...
    
    # Basescan API key for verification
    BASESCAN_API_KEY=your_api_key_here
    
    # Contract addresses (update after deployment)
    NEXT_PUBLIC_ESTORY_TOKEN_ADDRESS=
    NEXT_PUBLIC_STORY_PROTOCOL_ADDRESS=
    NEXT_PUBLIC_STORY_NFT_ADDRESS=
    NEXT_PUBLIC_VERIFIED_METRICS_ADDRESS=
    ```
    
    <Warning>
      **Security**: Never commit `.env.local` to Git. Use hardware wallets (Ledger/Trezor) for mainnet deployments.
    </Warning>
  </Step>
  
  <Step title="Get Basescan API Key">
    Register at [basescan.org/apis](https://basescan.org/apis) to get an API key for contract verification.
  </Step>
</Steps>

## Hardhat Configuration

The project uses Hardhat for contract compilation and deployment. Configuration is in `hardhat.config.ts`:

```typescript
import { HardhatUserConfig } from "hardhat/config";
import "@nomicfoundation/hardhat-toolbox";
import * as dotenv from "dotenv";

dotenv.config({ path: ".env.local" });

const config: HardhatUserConfig = {
  solidity: {
    version: "0.8.20",
    settings: {
      optimizer: {
        enabled: true,
        runs: 200,  // Optimized for frequent interactions
      },
    },
  },
  defaultNetwork: "baseSepolia",
  networks: {
    hardhat: {},
    baseSepolia: {
      url: "https://sepolia.base.org",
      accounts: process.env.ADMIN_WALLET_PRIVATE_KEY 
        ? [process.env.ADMIN_WALLET_PRIVATE_KEY] 
        : [],
      chainId: 84532,
    },
  },
  etherscan: {
    apiKey: {
      baseSepolia: process.env.BASESCAN_API_KEY || "",
    },
    customChains: [
      {
        network: "baseSepolia",
        chainId: 84532,
        urls: {
          // V2 API endpoint (fixes "Deprecated V1" error)
          apiURL: "https://api.etherscan.io/v2/api?chainid=84532",
          browserURL: "https://sepolia.basescan.org",
        },
      },
    ],
  },
  sourcify: {
    enabled: true,  // Enables open-source verification
  },
};

export default config;
```

<Info>
  The optimizer is set to 200 runs, balancing deployment cost and runtime efficiency. Increase to 1000+ for gas-optimized execution.
</Info>

## Compilation

Compile all Solidity contracts before deployment:

<Steps>
  <Step title="Clean Previous Build">
    ```bash
    npx hardhat clean
    ```
    
    Removes `artifacts/` and `cache/` directories.
  </Step>
  
  <Step title="Compile Contracts">
    ```bash
    npx hardhat compile
    ```
    
    **Expected output:**
    ```
    Compiled 12 Solidity files successfully (evm target: paris).
    ```
    
    Generated files:
    - `artifacts/contracts/*.sol/*.json` - Contract artifacts (ABI + bytecode)
    - `cache/` - Compilation cache for incremental builds
  </Step>
  
  <Step title="Verify Compilation">
    Check that all contracts compiled without errors:
    
    ```bash
    ls artifacts/contracts/
    ```
    
    You should see:
    - `EStoryToken.sol/`
    - `StoryProtocol.sol/`
    - `StoryNFT.sol/`
    - `PrivateVerifiedMetrics.sol/`
    - `VerifiedMetrics.sol/` (legacy)
  </Step>
</Steps>

<Warning>
  If compilation fails with "SPDX license identifier not provided", ensure all `.sol` files start with `// SPDX-License-Identifier: MIT`.
</Warning>

## Deployment Scripts

### Main Deployment (deploy.ts)

The primary deployment script deploys the core contracts:

```typescript
// scripts/deploy.ts
import { ethers } from "hardhat";

async function main() {
  const [deployer] = await ethers.getSigners();
  console.log("Deploying contracts with the account:", deployer.address);

  // 1. Deploy EStoryToken
  const EStoryToken = await ethers.getContractFactory("EStoryToken");
  const token = await EStoryToken.deploy(deployer.address);
  await token.waitForDeployment();
  const tokenAddress = await token.getAddress();
  console.log(`EStoryToken deployed to: ${tokenAddress}`);

  // 2. Deploy StoryProtocol
  const StoryProtocol = await ethers.getContractFactory("StoryProtocol");
  const protocol = await StoryProtocol.deploy(tokenAddress, deployer.address);
  await protocol.waitForDeployment();
  const protocolAddress = await protocol.getAddress();
  console.log(`StoryProtocol deployed to: ${protocolAddress}`);

  // 3. Deploy StoryNFT
  const StoryNFT = await ethers.getContractFactory("StoryNFT");
  const nft = await StoryNFT.deploy(deployer.address);
  await nft.waitForDeployment();
  const nftAddress = await nft.getAddress();
  console.log(`StoryNFT deployed to: ${nftAddress}`);

  console.log("\n--- SAVE THESE ADDRESSES ---");
  console.log("Go to 'lib/contracts.ts' and update the addresses now.");
}

main().catch((error) => {
  console.error(error);
  process.exitCode = 1;
});
```

<Note>
  This script does **not** deploy PrivateVerifiedMetrics (deployed separately due to CRE integration).
</Note>

### CRE Metrics Deployment (deployPrivateVerifiedMetrics.ts)

Deploy the privacy-preserving CRE receiver:

```typescript
// scripts/deployPrivateVerifiedMetrics.ts
import { ethers } from "hardhat";

// Chainlink KeystoneForwarder address on Base Sepolia
const BASE_SEPOLIA_FORWARDER = "0x82300bd7c3958625581cc2f77bc6464dcecdf3e5";

async function main() {
  const [deployer] = await ethers.getSigners();

  console.log("Deploying PrivateVerifiedMetrics with account:", deployer.address);
  console.log("KeystoneForwarder address:", BASE_SEPOLIA_FORWARDER);

  const PrivateVerifiedMetrics = await ethers.getContractFactory("PrivateVerifiedMetrics");
  const contract = await PrivateVerifiedMetrics.deploy(BASE_SEPOLIA_FORWARDER);
  await contract.waitForDeployment();

  const address = await contract.getAddress();
  console.log(`\nPrivateVerifiedMetrics deployed to: ${address}`);

  // Wait for confirmations
  const deployTx = contract.deploymentTransaction();
  if (deployTx) {
    await deployTx.wait(2);
  }

  console.log("\n--- NEXT STEPS ---");
  console.log("1. Update .env.local: NEXT_PUBLIC_VERIFIED_METRICS_ADDRESS=" + address);
  console.log("2. Update cre/iStory_workflow/config.staging.json verifiedMetricsAddress");
  console.log("3. Verify contract: npx hardhat verify --network baseSepolia " + address + " " + BASE_SEPOLIA_FORWARDER);
}

main().catch((error) => {
  console.error(error);
  process.exitCode = 1;
});
```

<Info>
  The KeystoneForwarder is Chainlink's on-chain component that receives signed DON reports. Address is fixed per network.
</Info>

### Troubleshooting Verification

**Error: "Compilation failed"**
- Ensure Solidity version matches deployed bytecode
- Check optimizer settings in `hardhat.config.ts`

**Error: "Already verified"**
- Contract is already verified, no action needed

**Error: "Invalid API key"**
- Verify `BASESCAN_API_KEY` in `.env.local`

## ABI Verification Script

The `verify-deployment.ts` script validates that deployed contracts match expected interfaces:

```typescript
// scripts/verify-deployment.ts
import { ethers } from "hardhat";

async function main() {
  const [signer] = await ethers.getSigners();
  console.log("Verifying deployments with account:", signer.address);

  const tokenAddress = process.env.NEXT_PUBLIC_ESTORY_TOKEN_ADDRESS;
  const protocolAddress = process.env.NEXT_PUBLIC_STORY_PROTOCOL_ADDRESS;
  const nftAddress = process.env.NEXT_PUBLIC_STORY_NFT_ADDRESS;

  let passed = 0;
  let failed = 0;

  // 1. Verify EStoryToken
  console.log("\n[1/3] Verifying EStoryToken at", tokenAddress);
  try {
    const token = await ethers.getContractAt("EStoryToken", tokenAddress);
    const name = await token.name();
    const symbol = await token.symbol();
    const maxSupply = await token.MAX_SUPPLY();

    console.log(`  Name: ${name}`);
    console.log(`  Symbol: ${symbol}`);
    console.log(`  Max Supply: ${ethers.formatEther(maxSupply)}`);

    if (name !== "eStoryToken") throw new Error(`Expected name 'eStoryToken', got '${name}'`);
    if (symbol !== "ESTORY") throw new Error(`Expected symbol 'ESTORY', got '${symbol}'`);
    if (maxSupply !== ethers.parseEther("100000000")) throw new Error("MAX_SUPPLY mismatch");

    console.log("  ✓ EStoryToken verified successfully");
    passed++;
  } catch (err: any) {
    console.error(`  ✗ EStoryToken verification failed:`, err.message);
    failed++;
  }

  // 2. Verify StoryProtocol
  console.log("\n[2/3] Verifying StoryProtocol at", protocolAddress);
  try {
    const protocol = await ethers.getContractAt("StoryProtocol", protocolAddress);
    const paused = await protocol.paused();
    console.log(`  Paused: ${paused}`);
    console.log("  ✓ StoryProtocol verified successfully");
    passed++;
  } catch (err: any) {
    console.error(`  ✗ StoryProtocol verification failed:`, err.message);
    failed++;
  }

  // 3. Verify StoryNFT
  console.log("\n[3/3] Verifying StoryNFT at", nftAddress);
  try {
    const nft = await ethers.getContractAt("StoryNFT", nftAddress);
    const name = await nft.name();
    const mintFee = await nft.mintFee();

    console.log(`  Name: ${name}`);
    console.log(`  Mint Fee: ${ethers.formatEther(mintFee)} ETH`);

    if (name !== "EStory Collections") throw new Error(`Expected name 'EStory Collections', got '${name}'`);

    console.log("  ✓ StoryNFT verified successfully");
    passed++;
  } catch (err: any) {
    console.error(`  ✗ StoryNFT verification failed:`, err.message);
    failed++;
  }

  console.log(`\nVerification complete: ${passed} passed, ${failed} failed`);
  if (failed > 0) process.exit(1);
}

main().catch((error) => {
  console.error(error);
  process.exitCode = 1;
});
```

<Steps>
  <Step title="Run ABI Verification">
    ```bash
    npx hardhat run scripts/verify-deployment.ts --network baseSepolia
    ```
    
    **Expected output:**
    ```
    [1/3] Verifying EStoryToken at 0xf9eDD76B...
      Name: eStoryToken
      Symbol: ESTORY
      Max Supply: 100000000.0
      ✓ EStoryToken verified successfully
    
    [2/3] Verifying StoryProtocol at 0xA51a4cA0...
      Paused: false
      ✓ StoryProtocol verified successfully
    
    [3/3] Verifying StoryNFT at 0x6D37ebc5...
    ```

  </Step>
</Steps>

## Mainnet Deployment Considerations

When deploying to Base Mainnet:

<Warning>
  **Security is critical**. Mainnet mistakes can result in permanent loss of funds.
</Warning>

<Steps>
  <Step title="Use Hardware Wallet">
    Never use `.env` private keys for mainnet. Use Ledger/Trezor via:
    
    ```bash
    npm install @nomicfoundation/hardhat-ledger
    ```
    
    Update `hardhat.config.ts`:
    
    ```typescript
    networks: {
      baseMainnet: {
        url: "https://mainnet.base.org",
        ledgerAccounts: ["0x..."],  // Your Ledger address
        chainId: 8453,
      },
    },
    ```
  </Step>
  
  <Step title="Audit Contracts">
    Have contracts audited by:
    - [OpenZeppelin](https://www.openzeppelin.com/security-audits)
    - [Trail of Bits](https://www.trailofbits.com/)
    - [Consensys Diligence](https://consensys.net/diligence/)
  </Step>
  
  <Step title="Test Extensively on Testnet">
    Run through all user flows:
    - Token transfers
    - Tipping
    - NFT minting
    - CRE verification
  </Step>
  
  <Step title="Deploy with Timelock">
    Use OpenZeppelin's `TimelockController` for 48-hour delay on admin actions:
    
    ```solidity
    TimelockController timelock = new TimelockController(
      2 days,           // Minimum delay
      new address[](0), // Proposers (set via grantRole)
      new address[](0), // Executors
      msg.sender        // Admin
    );
    ```
  </Step>
  
  <Step title="Set Up Multisig">
    Use Gnosis Safe for admin operations:
    
    1. Create Safe at [safe.global](https://safe.global/)
    2. Add 3-5 signers
    3. Require 3/5 signatures for admin actions
    4. Transfer `DEFAULT_ADMIN_ROLE` to Safe address
  </Step>
</Steps>

## Upgrading Contracts

Current contracts are non-upgradeable. To deploy new versions:

<Steps>
  <Step title="Deploy New Contract">
    ```bash
    npx hardhat run scripts/deploy-v2.ts --network baseSepolia
    ```
  </Step>
  
  <Step title="Migrate Data">
    Write migration script to transfer state from old to new contract.
  </Step>
  
  <Step title="Announce Deprecation">
    Give users 30-90 days notice before deprecating old contracts.
  </Step>
  
  <Step title="Update Frontend">
    Point `lib/contracts.ts` to new addresses. Consider supporting both during migration period.
  </Step>
</Steps>

<Info>
  Future versions may use UUPS or Transparent Proxy patterns for in-place upgrades without address changes.
</Info>

## Troubleshooting

### Common Deployment Errors

**Error: "insufficient funds for gas"**
- Solution: Add more ETH to deployer wallet

**Error: "nonce too low"**
- Solution: `npx hardhat clean && npx hardhat compile`

**Error: "contract creation code storage out of gas"**
- Solution: Reduce contract size or increase gas limit

**Error: "ProviderError: Too Many Requests"**
- Solution: Use a private RPC endpoint (Alchemy, Infura)
